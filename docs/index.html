Boas vindas!

O mini curso será uma introdução ao Quarto, a nova geração de RMarkdown!

O Quarto é um software de escrita técnica e científica que possibilita criar documentos reprodutíveis com R, como: listas de exercícios, relatórios, apresentações, sites simples apresentando as análises, entre outros.

Um importante avanço do Quarto em relação ao RMarkdown é não depender do R, portanto é possível usar com Python, por exemplo.
